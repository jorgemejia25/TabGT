# TabGT 1.1.0

Feature release focused on workspace awareness, shell integrations, and
inspector improvements.

## Highlights

- Workspace folder browser for local and SSH sessions.
- Detached terminal windows and cross-window tab movement.
- Inspector sections for workspace folders, Git status, and Claude Code state.
- Claude Code hook installation support and session activity tracking.
- Git prompt integration for branch, status, ahead/behind, and latest commit.
- Snippet launch modes for running commands in the current tab or a copied new
  tab.
- Improved SSH launch commands, diagnostics, retry handling, and Windows
  PowerShell integration.
- SSH private-key helper support for inline Keychain-backed keys.

## Security Notes

- This build is signed locally but is not notarized by Apple.
- macOS may require manual approval in System Settings > Privacy & Security on
  first launch.
- Passwords are stored in macOS Keychain.
- Private keys are referenced by path unless explicitly stored through the
  Keychain-backed inline-key flow.
- SSH host-key trust uses the system OpenSSH `known_hosts` flow.

## Install

Download `TabGT-1.1.0.dmg`, open it, and drag `TabGT.app` into Applications.
